# SDK 概述

艾塔达克提供多种语言的官方 SDK，帮助您快速集成 AI 能力到您的应用中。

## 官方 SDK

::card-group
  ::card{icon="i-simple-icons-python" title="Python SDK" to="/docs/sdk/python"}
  适用于 Python 3.8+ 的官方 SDK，功能完整，支持异步。
  ::

  ::card{icon="i-simple-icons-nodedotjs" title="Node.js SDK" to="/docs/sdk/nodejs"}
  适用于 Node.js 18+ 的官方 SDK，支持 TypeScript。
  ::
::

## 快速安装

::tabs
  ::div{label="Python"}
  ```bash
  pip install atdak
  ```
  ::

  ::div{label="Node.js"}
  ```bash
  npm install atdak
  # 或
  pnpm add atdak
  ```
  ::
::

## 快速开始

::tabs
  ::div{label="Python"}
  ```python
  from atdak import Atdak

  client = Atdak(api_key="YOUR_API_KEY")

  response = client.chat.completions.create(
      model="gpt-4o",
      messages=[
          {"role": "user", "content": "你好！"}
      ]
  )

  print(response.choices[0].message.content)
  ```
  ::

  ::div{label="Node.js"}
  ```javascript
  import { Atdak } from 'atdak';

  const client = new Atdak({ apiKey: 'YOUR_API_KEY' });

  const response = await client.chat.completions.create({
    model: 'gpt-4o',
    messages: [
      { role: 'user', content: '你好！' }
    ]
  });

  console.log(response.choices[0].message.content);
  ```
  ::
::

## 环境变量

所有 SDK 都支持通过环境变量配置 API Key：

```bash
export ATDAK_API_KEY="your-api-key"
```

这样在代码中就不需要显式传入 API Key：

```python
# Python - 自动读取环境变量
client = Atdak()
```

```javascript
// Node.js - 自动读取环境变量
const client = new Atdak();
```

## REST API

如果您使用的语言没有官方 SDK，可以直接调用 REST API：

```bash
curl -X POST https://api.atdak.com/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "你好！"}]
  }'
```
