# 欢迎使用艾塔达克

艾塔达克为开发者和企业提供强大、易用的 AI 能力，帮助您快速构建智能应用。

## 我们的服务

::card-group
  ::card{icon="i-lucide-message-square" title="大语言模型" to="/docs/api-reference/chat"}
  支持 GPT-4、Claude、Llama 等主流模型，提供文本生成、对话、翻译等能力。
  ::

  ::card{icon="i-lucide-image" title="图像智能" to="/docs/api-reference/vision"}
  图像识别、目标检测、OCR、图像生成等视觉 AI 能力。
  ::

  ::card{icon="i-lucide-mic" title="语音服务" to="/docs/api-reference/speech"}
  语音识别、语音合成、声纹识别，支持多语种。
  ::

  ::card{icon="i-lucide-cpu" title="边缘计算" to="/products/edge-pro"}
  本地部署 AI 推理，数据安全可控，毫秒级响应。
  ::
::

## 为什么选择艾塔达克？

- **简单易用**：标准 RESTful API，完善的 SDK 支持
- **稳定可靠**：99.9% 高可用保障
- **按量计费**：灵活的计费模式，降低使用门槛
- **数据安全**：企业级数据加密和隐私保护
