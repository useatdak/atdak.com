# API 参考

本文档提供艾塔达克所有 API 的详细说明。

## API 基础信息

| 项目 | 值 |
|------|-----|
| Base URL | `https://api.atdak.com/v1` |
| 认证方式 | Bearer Token |
| 请求格式 | JSON |
| 响应格式 | JSON |

## 请求格式

所有 API 请求应包含以下请求头：

```
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```

## 响应格式

### 成功响应

```json
{
  "id": "request-id",
  "object": "object-type",
  "created": 1701234567,
  "data": { ... }
}
```

### 错误响应

```json
{
  "error": {
    "code": "error_code",
    "message": "错误描述",
    "type": "error_type"
  }
}
```

## 错误码

| 错误码 | HTTP 状态 | 说明 |
|-------|----------|------|
| invalid_api_key | 401 | API Key 无效 |
| rate_limit_exceeded | 429 | 超出速率限制 |
| insufficient_quota | 402 | 配额不足 |
| invalid_request | 400 | 请求参数错误 |
| server_error | 500 | 服务器内部错误 |

## 速率限制

- 免费版：60 请求/分钟
- 专业版：600 请求/分钟
- 企业版：自定义

超出限制时返回 429 状态码，请求头包含：
- `X-RateLimit-Limit`: 限制值
- `X-RateLimit-Remaining`: 剩余次数
- `X-RateLimit-Reset`: 重置时间戳
