我们非常激动地宣布，艾塔达克首款 AI 边缘计算设备 **Edge Pro** 正式发布！

## 为什么需要边缘 AI？

在过去几年中，大语言模型展现出了惊人的能力，但云端部署模式也带来了一些挑战：

- **数据安全**：敏感数据需要上传到云端
- **网络延迟**：依赖网络连接，响应速度受限
- **成本控制**：大规模使用时 API 费用高昂
- **合规要求**：部分行业有严格的数据本地化要求

Edge Pro 正是为解决这些问题而生。

## Edge Pro 核心特性

### 强大算力
- 256 TOPS AI 算力
- 支持 7B-70B 参数模型
- 毫秒级推理响应

### 数据安全
- 完全本地化运行
- 数据不出企业网络
- 满足合规要求

### 开箱即用
- 预装 atdak OS
- 兼容 OpenAI API
- 10 分钟完成部署
