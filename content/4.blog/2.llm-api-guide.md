本指南将帮助您在 5 分钟内完成艾塔达克大语言模型 API 的接入。

## 准备工作

1. 注册艾塔达克账号
2. 在控制台获取 API Key
3. 选择合适的模型

## 快速开始

### 安装 SDK

```bash
# Python
pip install atdak-sdk

# Node.js
npm install @atdak/sdk
```

### 发送第一个请求

```python
from atdak import AtdakClient

client = AtdakClient(api_key="your-api-key")

response = client.chat.completions.create(
    model="atdak-gpt-4",
    messages=[
        {"role": "user", "content": "你好，请介绍一下自己"}
    ]
)

print(response.choices[0].message.content)
```

## 支持的模型

| 模型名称 | 上下文长度 | 特点 |
|---------|-----------|------|
| atdak-gpt-4 | 128K | 最强推理能力 |
| atdak-gpt-4-mini | 128K | 高性价比 |
| atdak-claude-3 | 200K | 超长上下文 |

## 最佳实践

1. **使用流式输出**：提升用户体验
2. **合理设置温度**：根据场景调整 temperature 参数
3. **利用系统提示**：通过 system 消息定制 AI 行为
