# v1.0.0 正式发布 🎉

我们很高兴地宣布艾塔达克 AI 云服务平台正式发布！

## 新功能

### AI 云服务
- 大语言模型 API（支持 GPT-4、Claude、Llama 等）
- 图像智能服务（识别、检测、OCR、生成）
- 语音服务（识别、合成、声纹识别）
- 文本向量化服务

### AI 硬件
- atdak Edge Pro 边缘计算设备发布
- 支持本地部署 70B 参数大模型
- 256 TOPS AI 算力

### 开发者工具
- Python SDK
- Node.js SDK
- 完整 API 文档
- 代码示例库
